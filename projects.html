Financial Dashboard

A financial dashboard built using Next.js to manage and track your finances. It incorporates a search function that allows users to look up financial records based on Customer, Email, Amount, Date, and Status, making it easier to find specific information within the dashboard. The app ensures secure authentication and authorization for protected pages using NextAuth.js, with a demo login system. The dashboard is responsive and optimized for both desktop and mobile devices.

Pokémon List

This is a web app I built to explore a list of Pokémon, displaying detailed stats and attributes for each one. It pulls real-time data from the Pokémon API, so the information is always up to date. I added a search function to quickly find specific Pokémon and pagination to make browsing easier. Users can click on a Pokémon to see more details.

Pac-Man Game

This project is a simple grid-based Pac-Man game that I built using HTML, CSS, and JavaScript. The game dynamically creates a 10x10 maze where Pac-Man moves through paths, avoids walls, and eats dots. The maze itself is generated using JavaScript with walls and paths set up in a 2D array. Players control Pac-Man using the arrow keys to navigate through the maze while avoiding obstacles. As Pac-Man moves, he eats the dots by passing over them, adding an interactive element to the game. The game also includes smooth animations where Pac-Man’s mouth opens and closes, making it more fun and engaging to play.
